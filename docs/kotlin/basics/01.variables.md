# Variables and Data Types in Kotlin

## Variable Declaration

Variables in Kotlin are declared using either:
- `val` for immutable **val**ues which can't be altered
- `var` for mutable **var**iables which can be changed

```kotlin run
val name = "Steve"     // Value is fixed (immutable)
var score = 0          // Variable can change (mutable)
```

## Variable Types

If you are assigning a value at the same time as declaring a variable, you don't need to specify the variable type, Kotlin will infer it.

But, if you are just declaring the variable, you must specify the type using `: Type`
```kotlin
val name: String               // String type specified
var age: Int                   // Int type specified
val birthPlace = "London"      // String type is inferred
var likesCats = true           // Boolean type is inferred
```

The most common types you will be using:
- `String`- for text
- `Char` - for single characters
- `Int` - for integer, whole numbers
- `Long` - for very large whole numbers
- `Double` - for decimal numbers
- `Boolean` - for true/false values

```kotlin
val name: String = "Steve"
val initial: Char = 'S'
var year: Int = 2026
val kmToNeptune: Long = 4500000000L      // Note the 'L' on the end
var cost: Double = 123.45
var isAlive: Boolean = true
```

Note:
 - `String` data (text) are wrapped with **double speech marks**, `"..."`
 - `Char` values are wrapped with **single speech marks**, `'.'`


## Variable Names

Variable names should:

- **Start with a lowercase letter**
- Use either *camelCase* (preferred) or *snake_case* for **multiple words** (be consistent and stick to one style!)

For example:

```kotlin
val firstName = "Steve"
var yearsOld = 10
val birthPlace = "London"
var likesCats = true
```
Alternatively (but don't do this!)...

```kotlin
val first_name = "Steve"
var years_old = 10
val birth_place = "London"
var likes_cats = true
```
