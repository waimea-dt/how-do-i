# Glossary of Terms


## Programming Fundamentals

### Basic Concepts

- algorithm

    Step-by-step instructions for solving a problem

- pseudocode

    Writing algorithm steps in plain English before coding

- flowchart

  A diagram showing the steps and decisions in an algorithm

- syntax

    The grammar rules of a programming language

- comment

    Text in code that explains what it does, ignored by the computer

- variable

    A named area of memory that holds a value like a number or text

- constant

    A named value that cannot be changed once set

- hard-coded

    Values written directly in code instead of using variables or user input

- parse

    To analyze and break down code or text into its parts to understand its structure

- parsing

    The process of analysing code or text to understand its structure


### Data Types

- bit

    The smallest unit of data - either 0 or 1

- byte

    8 bits - enough to store one character or a number from 0 to 255

- data type

    The kind of value a variable can hold, like number or text

- boolean

    A value that is either true or false

- char

    Character - a single letter, digit, or symbol in single quotes

- string

    Text - letters, words, and symbols in quotes

- concatenate

    To join strings together to make a longer string

- concatenation

    Joining strings together to make a longer string

- int

    Integer - whole numbers with no decimal point like 1, 42, or -5

- float

    Floating point number - numbers with decimal points, similar to Double

- double

    Numbers with decimal points like 3.14 or 2.5

- integer overflow

    When a number becomes too large for its data type and wraps around

- null

    A special value meaning nothing or empty

- literal value

    A fixed value written directly in code, like 42 or "hello"

- magic number

    A mysterious number in code that should be replaced with a named constant

- casting

    Converting a value from one data type to another

- mutable

    Something that can be changed

- immutable

    Something that cannot be changed once created


### Control Flow

- sequence

    Instructions executed one after another in order

- condition

    Something that is either true or false

- branch

    A point in code where different paths are chosen based on a condition

- selection

    Choosing which code to run based on a condition (if statements)

- if statement

    Chooses which code to run based on a condition, often paired with an else statement

- iteration

    One pass through a loop, or going through items in a collection one by one

- loop

    Code that repeats over and over until a condition is met

- for loop

    A loop that repeats a fixed number of times

- while loop

    A loop that repeats as long as a condition is true

- infinite loop

    A loop that never stops repeating

- recursion

    When a function calls itself to solve a problem

- boolean operator

    Combines conditions using AND, OR, or NOT

- AND

    Returns true only if both conditions are true

- OR

    Returns true if at least one condition is true

- NOT

    Reverses a condition from true to false or false to true

- logic gate

    An electronic component that performs Boolean operations like AND, OR, or NOT

- truth table

    A table showing all possible combinations of inputs and their outputs

- scope

    The area of code where a variable can be used

- syntax error

    An error in the grammar or spelling of code, caught before the program runs

- logic error

    When code runs without errors but produces incorrect results

- runtime error

    An error that happens while a program is running

- exception

    An error that happens while a program is running


### Functions & Procedures

- function

    A reusable block of code that does a specific job

- procedure

    A subroutine that performs actions but doesn't return a value

- subroutine

    Another name for a function or procedure - a reusable block of code

- parameter

    A placeholder in a function that receives a value

- argument

    The actual value you pass into a function

- return value

    A value passed back from a function


## Data Structures & Algorithms

### Data Structures

- data structure

    An organized way to store and access data

- array

    A list of items stored in order with numbered positions

- list

    A collection of items in a specific order

- record

    A data structure that groups related values together

- field

    One piece of data within a record

- node

    A point in a data structure, often containing data and links to other nodes

- graph

    A structure made of nodes connected by edges

- edge

    A connection between two nodes in a graph

- tree

    A hierarchical structure with a root and branches leading to child nodes

- binary tree

    A tree where each node has at most two children

- linked list

    A list where each item points to the next item in the chain

- stack

    A collection where the last item added is removed first (like a stack of plates)

- queue

    A collection where the first item added is removed first (like a queue of people)

- map

    A collection where each item has a name (key) and a value

- hash table

    A structure that stores data using key-value pairs for fast lookups


### Search & Sort Algorithms

- linear search

    Checking each item one by one until you find what you're looking for

- binary search

    Finding an item in a sorted list by repeatedly halving the search area

- bubble sort

    A simple sorting algorithm that swaps adjacent items if they're in wrong order

- insertion sort

    Building a sorted list by inserting each item in its correct position

- merge sort

    Dividing a list in half repeatedly, then merging sorted pieces back together

- quick sort

    Picking a pivot value and sorting items smaller and larger than it


### Algorithm Analysis

- complexity

    How long an algorithm takes or how much memory it uses as the input size grows

- big-O notation

    A way to describe how fast an algorithm runs as data grows

- tractability

    Whether a problem can be solved in a reasonable amount of time

- heuristic

    A practical rule-of-thumb that gives good-enough solutions quickly


### Computational Thinking

- computational thinking

    Problem-solving using techniques from computer science

- decomposition

    Breaking a complex problem into smaller, manageable parts

- abstraction

    Focusing on important details and ignoring irrelevant ones

- pattern recognition

    Spotting similarities or trends in data or problems

- algorithmic thinking

    Creating step-by-step instructions to solve a problem


## Object-Oriented Programming

- OOP

    Object-Oriented Programming - a way of organizing code using objects and classes

- object-oriented programming

    (OOP) a way of organizing code using objects and classes

- class

    A blueprint for creating objects with properties and behaviors

- object

    A specific thing created from a class

- property

    A variable that belongs to a class

- method

    A function that belongs to a class

- constructor

    Code that runs automatically when you create a new object

- encapsulation

    Hiding how something works inside a class

- inheritance

    When a class gets properties and methods from another class

- polymorphism

    When different types of objects respond to the same method differently

- interface

    A list of methods a class must have

- abstract class

    A class that's only used as a template for other classes


## Software Development

### Development Practices

- bug

    An error or flaw in a program

- debugging

    Finding and fixing errors in your code

- breakpoint

    A marker that pauses program execution for debugging

- testing

    Checking if code works correctly before releasing it

- unit testing

    Testing individual functions or small pieces of code in isolation

- integration testing

    Testing how different parts of a program work together

- alpha testing

    Early testing by developers before releasing to users

- beta testing

    Testing by real users before official release

- black-box testing

    Testing software without knowing how it works internally

- white-box testing

    Testing software with knowledge of its internal structure

- dry-run

    Manually working through code on paper to check it works

- trace table

    A table showing how variables change as code runs step-by-step

- documentation

    Written explanations of what code does and how to use it

- refactoring

    Improving code to make it cleaner without changing what it does

- validation

    Checking that data is correct and safe before using it

- accessibility

    Making software usable by people with disabilities

- usability

    How easy software is to use

- user experience

    (UX) How a person feels when using software

- UX

    User Experience - How a person feels when using software


### Development Methodologies

- requirements specification

    A document describing what software must do

- stakeholder

    A person with an interest in the project, like clients or users

- waterfall

    A sequential development approach where each phase must finish before the next

- agile

    A flexible development approach that delivers software in small iterations

- scrum

    An Agile framework using sprints, daily meetings, and specific roles

- sprint

    A short period (usually 2-4 weeks) in Agile development

- prototype

    An early version of software used to test ideas


### Version Control & Tools

- version control

- repository

    A storage location for code and its history

- repo

    Short for repository - a storage location for code and its history

- Git

    A version control system for tracking changes to code over time

- GitHub

    A website for sharing and collaborating on code using Git

- GitHub Pages

    A free service from GitHub for hosting websites directly from your repository

- translate

- compiling

- compiler

- interpreting

    The process of converting and running code line-by-line as the program executes

- interpreter

    Converts and runs code line-by-line while the program is running

- IDE

    Integrated Development Environment - an app for writing and testing code all in one place

- IntelliJ

    Short for IntelliJ IDEA - a popular IDE for Java and Kotlin development

- IntelliJ IDEA

    A powerful IDE made by JetBrains for Java and Kotlin programming

- VSCode

    Short for Visual Studio Code - a popular free code editor made by Microsoft

- Visual Studio Code

    A free, lightweight code editor made by Microsoft with support for many languages

- JetBrains

    A company that makes professional developer tools like IntelliJ IDEA and PyCharm

- Gradle

    A build system used to compile, test, and package Java and Kotlin projects

- build system

    A tool that automates compiling, testing, and packaging your code into a runnable program

- JDK

    Java Development Kit - the tools needed to write and run Java programs

- JVM

    Java Virtual Machine - the platform that runs Java and Kotlin programs

- API

    Application Programming Interface - a way for programs to talk to each other

- open-source

    Software where anyone can view and modify the code


### User Interfaces

- UI

    User Interface - how a person interacts with software

- CLI

    Command Line Interface - typing text commands instead of clicking buttons

- TUI

    Text User Interface - a text-based menu system in the terminal

- GUI

    Graphical User Interface - windows, buttons, and menus you click on screen

- wireframe

    A basic visual layout showing where elements go in a user interface

- Swing

    A Java library for creating windows and buttons in desktop apps


## Computer Science Theory

### Data Representation

- bit

    The smallest unit of data - either 0 or 1

- binary

    Base-2 number system using only 0s and 1s

- denary

    Base-10 number system - normal counting using digits 0-9

- decimal

    Another name for denary - base-10 number system

- hexadecimal

    Base-16 number system using digits 0-9 and A-F, often used for color codes

- two's complement

    A method for representing negative numbers in binary

- bit-shifting

    Moving bits left or right to multiply or divide by powers of 2

- floating-point

    A way of storing decimal numbers in binary using scientific notation

- ASCII

    American Standard Code for Information Interchange - represents text as numbers 0-127

- Unicode

    A modern code for representing text in all languages and symbols

- pixel

    The smallest dot of color on a screen

- resolution

    The number of pixels in an image (width × height)

- colour-depth

    The number of bits used to represent the color of each pixel

- bitmap

    An image made of pixels, gets blurry when enlarged

- vector graphics

    An image made of mathematical shapes, stays sharp when enlarged

- sampling rate

    How many times per second audio is measured when digitizing sound

- compression

    Reducing file size by removing redundant data

- lossless

    Compression that reduces file size without losing any data

- lossy

    Compression that reduces file size by removing some data permanently

- run-length encoding

    A compression method that replaces repeating data with a count

- Huffman coding

    A compression method that uses shorter codes for common data

- metadata

    Data about data, like the date a photo was taken


### Low-Level Computing

- machine code

    Binary instructions the CPU can execute directly

- assembly

    Low-level programming language close to machine code

- cache

    Fast temporary storage for frequently used data

- buffer

    Temporary storage while data moves between two places


## Security & Privacy

- authentication

    Proving who you are, usually with a username and password

- password

    A secret word or phrase used to access an account

- two-factor authentication

    (2FA) Using two methods to prove your identity, like password plus phone code

- 2FA

    Two-Factor Authentication - Using two methods to prove your identity, like password plus phone code

- encryption

    Scrambling data so only authorized people can read it

- decryption

    Unscrambling encrypted data back to readable form

- symmetric encryption

    Encryption where the same key encrypts and decrypts data

- asymmetric encryption

    Encryption using a public key to encrypt and a private key to decrypt

- public key

    A key anyone can use to encrypt data, but only you can decrypt it

- private key

    A secret key that decrypts data encrypted with your public key

- certificate

    A digital document that proves a website or person's identity

- digital signature

    A way to prove a message really came from you using encryption

- cypher

    An algorithm for encrypting or decrypting data

- malware

    Harmful software designed to damage or spy on computers

- virus

    Malware that spreads by copying itself to other files

- brute-force attack

    Trying every possible password until finding the right one

- phishing

    Fake messages that trick people into revealing personal information

- social engineering

    Tricking people into revealing passwords or sensitive information

- cross-site scripting

    (XSS) Injecting malicious code into a website to attack users

- XSS

    Cross-Site Scripting - Injecting malicious code into a website to attack users

- DDoS

    Distributed Denial of Service - overwhelming a server with traffic to crash it

- database

    An organized collection of data stored on a computer

- relational database

    A database that stores data in tables linked by relationships

- table

    Data arranged in rows and columns like a spreadsheet

- row

    A single entry in a database table containing related data

- column

    A category of data in a database table, like Name or Age

- primary key

    A unique identifier for each row in a table

- foreign key

    A column that links to a primary key in another table

- index

    A structure that speeds up searching in a database

- normalisation

    Organizing database tables to reduce redundancy and improve consistency

- query

    A question you ask to get data from a database

- SQL

    Structured Query Language - a language for asking questions and managing databases

- CRUD

    Create, Read, Update, Delete - the four basic database operations

    A security attack that inserts malicious SQL code into a query

- WAN

- dynamic IP

    An IP address that changes each time you connect

- MAC address

    Media Access Control address - a unique identifier for network hardware

- DNS

    Domain Name System - the system that converts website names into IP addresses

- DHCP

    Dynamic Host Configuration Protocol - automatically assigns IP addresses to devices

- NAT

    Network Address Translation - lets multiple devices share one public IP address

- Ethernet

    A wired connection standard for local networks using cables

- wi-fi

    Wireless technology for connecting to a network

- fibre

    Fibre-optic cables that use light to transmit data at very high speeds

- TCP

    Transmission Control Protocol - ensures data is sent reliably over a network

- HTTP

    HyperText Transfer Protocol - the system used to send web pages across the internet

- HTTPS

    HyperText Transfer Protocol Secure - a secure encrypted version of HTTP

- SSL

    Secure Sockets Layer - encrypts data sent over the internet

- TLS

    Transport Layer Security - encrypts data sent over networks

- URL

    Uniform Resource Locator - the web address you type in your browser

- packet

    A small chunk of data sent across a network

- port

    A numbered channel for different types of network communication

- router

    A device that directs network traffic between different networks

- modem

    A device that connects your home network to the internet

- firewall

    Security that blocks unauthorized network access

- bandwidth

    How much data can travel through a network connection

- bps

    Bits per second - the basic measure of data transfer speed

- kbps

    Kilobits per second - thousands of bits per second

- Mbps

    Megabits per second - millions of bits per second

- Gbps

    Gigabits per second - billions of bits per second
