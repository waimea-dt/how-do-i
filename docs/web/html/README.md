# HTML

**HTML** (HyperText Markup Language) is the building block of every webpage. It defines the structure of a page - the headings, paragraphs, images, and links that make up what you see in a browser.

```html
<h1>Hello, World!</h1>
<p>This is my first webpage.</p>
```

> [!TIP]
> Use the sidebar to navigate the topics in this guide. If you're new to HTML, start with **HTML Introduction** and work through the pages in order.


## What is HTML For?

HTML is not a programming language - it's a **markup language**. It tells the browser what things *are* (a heading, a paragraph, an image) rather than what to *do*.

- **Structure** - organise content into headings, sections, lists, tables
- **Content** - embed text, images, video, and links
- **Semantics** - give meaning to content so browsers and screen readers understand it

> [!NOTE]
> HTML works hand-in-hand with CSS (for styling) and JavaScript (for interactivity). Together they form the core of web development.
