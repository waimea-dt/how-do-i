# JavaScript

**JavaScript** is a programming language that runs in the browser. It makes webpages interactive - responding to clicks, updating content, animating elements, and communicating with servers.

```javascript
document.querySelector('h1').textContent = 'Hello!'
```

> [!TIP]
> Use the sidebar to navigate the topics in this guide. If you're new to JavaScript, start with **JavaScript Introduction**.


## What is JavaScript For?

JavaScript is the only programming language that runs natively in web browsers. It lets you:

- **React to events** - clicks, key presses, form submissions
- **Modify the page** - add, remove, or change HTML and CSS dynamically
- **Communicate with servers** - fetch data without reloading the page
- **Build applications** - games, tools, and full web apps

> [!NOTE]
> JavaScript is one of the most widely used programming languages in the world, and is essential for modern web development.
